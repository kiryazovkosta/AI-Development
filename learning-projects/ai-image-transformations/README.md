# mondrAIn

Playful desktop app for creative image transformations (exploration-first).

## Run (Windows)

- Build: `dotnet build`
- Run: `dotnet build -t:Run -f net10.0-windows10.0.19041.0`

## Manual verification (current MVP)

1. Launch the app.
2. Confirm the home screen shows the title **mondrAIn**.
3. Click **Upload** and pick an image.
4. Confirm the selected image renders in the working area.
